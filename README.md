# Editkaro Portfolio Website

## 📌 Project Overview

This is a responsive portfolio website developed for **Editkaro**, a creative video editing and social media marketing agency.

## 🚀 Features

* Responsive Design
* Home Page
* Portfolio Page
* About Page
* Contact Page
* Contact Form connected to Google Sheets using Google Apps Script
* Modern UI with HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Apps Script
* Google Sheets
* Git & GitHub

## 📂 Project Structure

* index.html
* portfolio.html
* about.html
* contact.html
* css/
* js/
* images/
* videos/
